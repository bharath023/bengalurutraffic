Signal Delay Compounding Formula
// Core compounding equation
delay_per_trip = delay_per_signal × signals_per_km × commute_km     // seconds
annual_loss = delay_per_trip × 2 (round trip) × 250 (working days) / 60  // hours

// Economic cost
time_value = annual_loss_hrs × (annual_salary / (250 × 8))             // ₹ per year
fuel_cost = annual_loss_hrs × idle_fuel_rate × petrol_price          // ₹ per year
total_cost = time_value + fuel_cost
